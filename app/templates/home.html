Tutoring app

Collaboration white board and chat, saving drawings as images to help me with private lessons. Soon will add voice chat and pasting images. There is also integration with facebook api to send me reminders from my fanpage to my private facebook account. Automatic private tutoring room creation with url with token.

Stack used:

  1. Django
  2. Django Channels
  3. WebSockets
  4. Redis
  5. Celery
  6. PostgreSQL
  7. JavaScript
  8. Bootstrap
